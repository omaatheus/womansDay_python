# Womans Day 

Um Programa para calcular descontos em homenagem ao dia das mulheres.

## Funcionalidades

- **FUNÇÕES**
  - Descontos;
  - Caso seja mulher, recebe 20% de desconto.
  - Caso seja homem e casado, recebe 5%.
